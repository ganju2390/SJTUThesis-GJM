| **1.1.2 动力学方程**                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 通过调研 Fossen 提出的矢量化建模框架，明确了 ROV 在六自由度（6-DOF）下的完整动力学描述。该模型能够全面捕获水下运动的非线性特征，其标准矢量表达式如下：                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$M\overset{˙}{\nu} + C(\nu)\nu + D(\nu)\nu + g(\eta) = \tau$$                                                                                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 方程中的各参数的物理含义与数学性质如下：                                                                                                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 惯性项$M$：除了机器人自身的质量矩阵$M_{RB}$，水下运动最显著的特征是附加质量$M_{A}$。当 ROV 在水中加速时，必须带动周围的一部分流体共同运动，这在物理上表现为惯性增大。对于小型 ROV，附加质量往往能达到自身质量的 50% 以上，是建模中不可忽视的项。                                                                                                                                                                                                                  |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 向心力与科氏力项$C(\nu)$：该项描述了旋转坐标系中的非线性耦合效应。在高速转向或多自由度联动时，这一项会导致轴向间的动力学干扰。                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 水动力阻尼项$D(\nu)$：ROV 在水中受到的阻尼主要由两部分组成：低速时的线性粘性阻尼和高速时的非线性二次阻尼。由于本项目涉及的 ROV 结构具有较高的对称性，在简化模型中主要考虑对角线方向的主阻尼，忽略复杂的交叉耦合阻尼。                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 恢复力项$g(\eta)$：由重力与浮力共同构成。本设计采用"深重心"方案，即重心（CG）严格位于浮心（CB）下方，利用静水力矩实现机器人的自稳，这使得 ROV 在无控制状态下也能保持水平姿态。                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.1.2.1惯性矩阵**$\mathbf{M}$                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

| M_{\dot{u}} & M_{\dot{v}} & M_{\dot{w}} & M_{\dot{p}} & M_{\dot{q}} & M_{\dot{r}} \\                                                                                                                                                                                                                                                                                                                                                                              |

| \end{bmatrix} = \begin{bmatrix}                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| A_{11} & A_{12} \\                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| A_{21} & A_{22}                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| \end{bmatrix}$$                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 其中$A_{11}$为平动带来的附加质量力，$A_{22}\$为旋转带来的附加质量力矩，$A_{12}$代表旋转导致附加质量力，$A_{21}$为平动带来的附加质量力矩。$M_{A}$矩阵的6×6展开形式中的元素为该方向上的力/力矩对指定运动方向的加速度/角加速度的求导。                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.1.2.2科里奥利与向心力矩阵**$\mathbf{C(\nu)}$                                                                                                                                                                                                                                                                                                                                                                                                                  |

| 0 & - a_{3} & a_{2} \\                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| a_{3} & 0 & - a_{1} \\                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|  - a_{2} & a_{1} & 0                                                                                                                                                                                                                                                                                                                                                                                                                                              |

|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 阻尼矩阵通常被定义为线性阻尼和二次非线性阻尼的和：                                                                                                                                                                                                                                                                                                                                                                                                                |

|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 2.  $\mathbf{X}_{\mathbf{u}\left| \mathbf{u} \right|}$**：**二次方阻尼导数（Quadratic hydrodynamic derivative）。                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 3.  $\left| \mathbf{u} \right|$**：**速度的绝对值，保证阻尼力始终与运动方向相反（消耗能量）。                                                                                                                                                                                                                                                                                                                                                                     |

| 恢复力由重力$W\  = \ mg$和浮力$B = \rho g\nabla$产生。假设重心坐标为$\left( x_{g},y_{g},z_{g} \right)$，浮心坐标为$\left( x_{b},y_{b},z_{b} \right)$：                                                                                                                                                                                                                                                                                                            |

| 为了提高控制器在嵌入式系统STM32F4上的实时运算效率，在建模时作如下合理假设：                                                                                                                                                                                                                                                                                                                                                                                       |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 1.  低速运动假设： 假设 ROV 工作在低速巡检状态（$< 0.5m/$），此时可以忽略高阶流体干扰。                                                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

| 3.  流体环境平稳： 初始建模假设环境流场恒定，将随之而来的时变海流扰动视为系统的外部未建模扰动，并计划通过后续的控制算法（如滑模控制的鲁棒性）进行补偿。                                                                                                                                                                                                                                                                                                           |

| **1.2.1惯性矩阵**$\mathbf{M}$**的化简**                                                                                                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 在建立 ROV 的动力学模型时，惯性矩阵$M$是描述机器人运动惯性的核心参数。它由刚体惯性矩阵$M_{RB}$和附加质量矩阵$M_{A}$两部分组成：$M = M_{RB} + M_{A}$。为了满足控制算法的实时性要求，本项目在硬件设计阶段便采取了多项优化措施，使得复杂的惯性矩阵得以最终化简为常数对角矩阵。                                                                                                                                                                                       |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.2.1.1 刚体惯性矩阵**$\mathbf{M}_{\mathbf{RB}}$**的化简过程**                                                                                                                                                                                                                                                                                                                                                                                                  |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 刚体惯性矩阵的标准形式包含质量$m$、重心坐标$r_{g} = \lbrack x_{g},y_{g},z_{g}\rbrack^{T}$以及惯性张量 $I\_ b$。                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| （1）质心与原点的重合设计（$r_{g} \approx 0$）                                                                                                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 在硬件布局阶段，通过对电池组、耐压舱及配重块的精确排布，使机器人的重心与体坐标系的原点基本重合。此时$r_{g} = \lbrack 0,0,0\rbrack^{T}$时，由反对称矩阵算子$S(r_{g})$产生的平移与旋转耦合项全部消失。原本极其复杂的交叉项$M_{12} = - mS(r_{g})$和$M_{21} = mS(r_{g})$变为零矩阵,可得：                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$M_{RB} = \lbrack\begin{matrix}                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| mI_{3 \times 3} & 0_{3 \times 3} \\                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 0_{3 \times 3} & I_{b}                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| \end{matrix}\rbrack$$                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| （2）质量均匀分布与结构对称性($I_{xy},I_{xz},I_{yz} \approx 0$)                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| ROV 采用四角对称的 8 推进器矢量布局，且结构件（框架、浮力材）左右及前后对称。这种高度的几何对称性确保了质量分布的均匀，使得惯性积极小，可以忽略不计。因此惯性张量简化为仅包含主轴转动惯量的对角阵：                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$I_{b} = diag\{ I_{x},I_{y},I_{z}\}$$                                                                                                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.2.1.2附加质量矩阵**$\mathbf{M}_{\mathbf{A}}$**的化简过程**                                                                                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 附加质量描述了水下运动时带动周围流体产生的惯性效应。这里假设流体动力学时对称的。因为ROV 具有三个相互垂直的对称平面（水面、横剖面、纵剖面），在流体动力学中，不同自由度之间的附加质量耦合作用极弱。例如，在 $x$方向加速时，产生的$y$方向附加力矩几乎为零。这里参考 Fossen 对低速、对称潜水器的经验处理，忽略非对角线上的流体力导数。最终化简得：                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$M_{A} = - diag\{ X_{\overset{˙}{u}},Y_{\overset{˙}{v}},Z_{\overset{˙}{w}},K_{\overset{˙}{p}},M_{\overset{˙}{q}},N_{\overset{˙}{r}}\}$$                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.2.1.3最终化简：常数对角惯性矩阵**$\mathbf{M}$                                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$M = M_{RB} + M_{A} = diag\{ m - X_{\overset{˙}{u}},m - Y_{\overset{˙}{v}},m - Z_{\overset{˙}{w}},I_{x} - K_{\overset{˙}{p}},I_{y} - M_{\overset{˙}{q}},I_{z} - N_{\overset{˙}{r}}\}$$                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 最终，我们得到了一个对角惯性矩阵，能够很直观地求出逆矩阵，方便后续动力学方程计算。                                                                                                                                                                                                                                                                                                                                                                                |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.2.2科里奥利与向心力矩阵**$C(\nu)$ **的处理**                                                                                                                                                                                                                                                                                                                                                                                                                  |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **刚体项**$\mathbf{C}_{\mathbf{RB}}\mathbf{(\nu)}$**的化简：**                                                                                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 科里奥利与向心力矩阵描述了在旋转体坐标系下，由于线速度与角速度耦合产生的惯性力。在Fossen的原始模型中，它由刚体部分和附加质量部分组成，且表现出极强的非线性。                                                                                                                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 本项目研发的 ROV 主要执行近距离的精细抓取与巡检任务。科里奥利力的大小与速度的平方（$\nu^{2}$）成正比。在低速运动工况下，该项的量级远小于推进器提供的推力和流体阻力，对系统动力学响应的影响可以忽略。同时由于在硬件设计时已确保重心与几何中心重合，刚体科氏力矩阵中由$M_{12}、M_{21}$引起的复杂非线性项在物理层面被消除，进一步减弱了旋转与平移之间的动力学耦合。因此在这里我们将其忽略，设为零矩阵。                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **附加质量项**$\mathbf{C}_{\mathbf{A}}\mathbf{(\nu)}$**的处理：**                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 附加质量是时变非线性的，难以处理，因此将此项归为外部扰动，作为后续扰动观测器工作内容。                                                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **1.2.3阻尼矩阵**$D(\nu)$**的线性化与对角化处理**                                                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **线性化：**本项目ROV主要以低速执行水下精准定位或机械臂抓取任务时，流体动力学中的雷诺数较低。此时，粘性线性阻尼占主导地位，而与速度平方成正比的二次阻尼（Drag force）对系统动力学的影响显著下降，可以忽略不计。所以这里消去阻力矩阵的非线性项，同时也方便后续的系统阻尼参数辨识。                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **对角化：**本项目设计的ROV具有明显的三个对称平面。在流体力学中，对称结构在单一方向运动时诱导的其他自由度阻力极小。从系统辨识的角度出发，完整的6x6阻尼矩阵包含36个未知参数，通过实验标定几乎不可能实现。简化为对角阵后，仅需针6个主自由度（$x,y,z,roll,pitch,yaw$）进行阶跃响应实验即可获得准确参数。因此假设不同自由度之间的运动是相互独立的，将阻尼矩阵简化为对角矩阵。                                                                                         |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 最终化简得：                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$D = \begin{bmatrix}                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| d_{u} & 0 & 0 & 0 & 0 & 0 \\                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 0 & d_{v} & 0 & 0 & 0 & 0 \\                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 0 & 0 & d_{w} & 0 & 0 & 0 \\                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 0 & 0 & 0 & d_{p} & 0 & 0 \\                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 0 & 0 & 0 & 0 & d_{q} & 0 \\                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 0 & 0 & 0 & 0 & 0 & d_{r}                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| \end{bmatrix}$$                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

| **2 控制算法的设计与研发**                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 本项目面向浅水复杂流场、低速高精度作业场景，以六自由度简化动力学模型为基础，构建分层级、双闭环、带模型前馈的运动控制架构，实现 ROV 横滚 / 俯仰≤±2°、定深≤±10cm 的高精度控制目标。控制算法围绕姿态稳定、速度跟踪、前馈补偿、推力分配四大核心环节展开，兼顾低速作业柔顺性、扰动鲁棒性与工程可实现性。                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 浅水水域存在洋流、紊流、线缆张力、海草缠绕等强非线性扰动，传统 PID 在大误差易饱和、小误差响应慢、耦合扰动抑制弱。本算法设计遵循三原则：                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 1.  解耦控制：利用 ROV 对称结构，将 6 自由度拆分为姿态 3 自由度（Roll/Pitch/Yaw）与平移3自由度（Surge/Sway/Heave）独立控制，降低耦合干扰。                                                                                                                                                                                                                                                                                                                        |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 2.  分级闭环：姿态采用位置-角速度双闭环，兼顾稳态精度与动态响应；平移采用速度单闭环，适配手动操控与自动定深/定高需求。                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 3.  模型补偿：融入动力学前馈，提前抵消惯性、阻尼、恢复力的固有影响，降低反馈控制器负担，提升响应速度与抗扰能力。                                                                                                                                                                                                                                                                                                                                                  |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 算法整体架构：传感器数据采集→姿态 / 速度解算→误差计算→闭环控制→模型前馈补偿→总力 / 力矩计算→推力分配→执行器输出。                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **2.1 姿态控制：平方根-PID双环结构 (SQRT-PID)**                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 针对姿态自由度（Roll, Pitch, Yaw），采用双环结构。其中第一环引入平方根函数，误差较大时限制角速度，防止控制饱和，并在误差小时保持灵敏度。                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **第一环：角度位置环**                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 计算期望姿态角$\eta_{att,d} = \lbrack 0,0,\psi_{d}\rbrack^{T}$与当前姿态角$\eta_{att} = \lbrack\phi,\theta,\psi\rbrack^{T}$的误差：                                                                                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$e_{\eta} = \eta_{att,d} - \eta_{att}$$                                                                                                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 通过比例增$K_{p}$结合平方根映射，得到期望角速度$\eta_{att,d}$：                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$\nu_{att,d} = K_{att} \cdot \text{sgn}(e_{\eta})\sqrt{|e_{\eta}|}$$                                                                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 大误差时：$\sqrt{|e_{\eta}|}$增长慢于线性项，限制角速度峰值，防止超调；                                                                                                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 小误差时：$\sqrt{|e_{\eta}|}$保留高分辨率，保证微调精度。                                                                                                                                                                                                                                                                                                                                                                                                         |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $K_{att}$为姿态环比例增益，通过仿真与实验整定。                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **第二环：角速度环**                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 计算期望角速度$\nu_{att,d}$与当前采样角速度$\nu_{att} = \lbrack p,q,r\rbrack^{T}$的偏差，通过 PID 计算期望反馈力矩$\tau_{att,fb}$：                                                                                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$e_{\nu} = \nu_{att,d} - \nu_{att}$$                                                                                                                                                                                                                                                                                                                                                                                                                             |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$\tau_{att,fb} = K_{P}e_{\nu} + K_{I}\int e_{\nu}dt + K_{D}\frac{de_{\nu}}{dt}$$                                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **2.2 平移速度控制：单层 PID**                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 对于平移自由度（Surge, Sway, Heave），直接通过手柄或设定值给定期望速度$\nu_{pos,d} = \lbrack u_{d},v_{d},w_{d}\rbrack^{T}$，计算反馈力$\tau_{pos,fb}$：                                                                                                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$e_{pos} = \nu_{pos,d} - \lbrack u,v,w\rbrack^{T}$$                                                                                                                                                                                                                                                                                                                                                                                                              |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$\tau_{pos,fb} = K_{P,pos}e_{pos} + K_{I,pos}\int e_{pos}dt + K_{D,pos}\frac{de_{pos}}{dt}$$                                                                                                                                                                                                                                                                                                                                                                     |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **2.3 模型前馈补偿 (Model Feedforward)**                                                                                                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 基于之前化简的 Fossen 动力学模型，前馈项$\tau_{ff}$的目的是预先抵消机器人由于自身物理特性产生的力/力矩。                                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| $$\tau_{ff} = \overset{\hat{}}{M}{\overset{˙}{\nu}}_{d} + \overset{\hat{}}{D}\nu + \overset{\hat{}}{g}(\eta)$$                                                                                                                                                                                                                                                                                                                                                    |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 各前馈项物理意义与实现方式：                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 1.  惯性前馈$\overset{\hat{}}{M}{\overset{˙}{\nu}}_{d}$:补偿机器人加减速时的刚体惯性与附加质量，避免启动滞后、停止过冲。                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 2.  阻尼前馈$\overset{\hat{}}{D}\nu$:补偿水下线性阻尼，抵消流体阻力，保证速度指令与实际输出一致。                                                                                                                                                                                                                                                                                                                                                                 |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 3.  恢复力前馈$\overset{\hat{}}{g}(\eta)$:补偿重力 / 浮力差与姿态带来的静恢复力矩，尤其提升 Heave 与 Roll/Pitch 稳定性。                                                                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| 通过补充控制前馈，能够大幅降低反馈控制器输出负担，提速系统响应，实现对模型已知扰动实现完全补偿，提升低速控制精度，同时为后续扰动观测器预留设计空间，只处理未知外部扰动。                                                                                                                                                                                                                                                                                          |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **2.4推力分配**                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

| **存在的问题及解决思路：**                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                      |
| 在现阶段的系统设计与实验验证过程中，虽然初步实现了 ROV 的 6 自由度闭环控制与远程作业功能，但受限于实验条件与物理建模的深度，系统仍存在以下待优化的问题及相应的解决思路。                                                             |
|                                                                                                                                                                                                                                      |
| **1 动力学参数辨识的精确度问题**                                                                                                                                                                                                     |
|                                                                                                                                                                                                                                      |
| 存在问题：目前系统建立的动力学模型中，阻尼矩阵的六个自由度参数主要依赖于理论估算与经验值。由于水下环境的复杂性，流体阻力具有明显的非线性特征，参数的不准确直接导致了动力学前馈输出的偏差，使得控制器在高速运动或复杂工况下表现欠佳。 |
|                                                                                                                                                                                                                                      |
| 解决思路：                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                      |
| > 后续拟开展专项的水下阻尼系数辨识实验。通过控制 ROV 在特定自由度上执行阶跃响应与减速停机实验，利用高频传感器数据，结合最小二乘法进行离线参数回归。                                                                                  |
|                                                                                                                                                                                                                                      |
| **2 复杂流场扰动与非线性力的补偿**                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                      |
| 存在问题：当前的控制策略忽略了科里奥利力的影响，且对外部涌浪、暗流等随机扰动缺乏专门的处理机制，目前仍完全依赖 PID 反馈回路的鲁棒性进行硬性弥补。在强扰动环境下，控制器的超调量与稳态误差会显著增加。                                |
|                                                                                                                                                                                                                                      |
| 解决思路：                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                      |
| - 扰动观测器（DOB）集成：在现有的架构中引入扰动观测器，实时估计建模误差与外部扰动。                                                                                                                                                  |
|                                                                                                                                                                                                                                      |
| - 先进控制算法应用：研究并引入滑模控制（SMC）或鲁棒控制策略。利用滑模面对非线性扰动的不敏感性，确保系统在动力学模型不确定及外部水流干扰下的渐进稳定性。                                                                              |
|                                                                                                                                                                                                                                      |
| **3 样机阶段向工程实机的过渡验证**                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                      |
| 存在问题：目前的嵌入式代码调试与控制逻辑验证主要在原理样机上完成，尚未在最终设计的工程化 ROV上进行大规模闭环测试。样机与实机在质量分布、惯性矩及水力外形上的差异可能导致控制参数失效。                                               |
|                                                                                                                                                                                                                                      |
| 解决思路：                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                      |
| - 联合仿真环境建设：利用MATLAB建立项目 ROV 的动力学模型，进行控制算法仿真。                                                                                                                                                          |
|                                                                                                                                                                                                                                      |
| - 实机迭代实验：在完成项目 ROV 的总装后，通过 V5_SUBBuilder 快速迁移软件框架，针对实机的物理特性重新调整控制律参数，完成从池塘实验到开放水域实验的跨越。                                                                             |
|                                                                                                                                                                                                                                      |
| **4 执行机构推力映射的非线性修正**                                                                                                                                                                                                   |
|                                                                                                                                                                                                                                      |
| 存在问题：当前对推力器的控制采用简单的线性映射映射方式，即将期望推力直接归一化为油门信号分发给驱动器。实际上，水下推力器的转速与推力之间呈非线性关系，且正反转推力存在不对称性，这限制了动力分配的精准度。                           |
|                                                                                                                                                                                                                                      |
| 解决思路：                                                                                                                                                                                                                           |
|                                                                                                                                                                                                                                      |
| - 推力特性表构建：利用静推力测试台采集电机在全量程下的推力数据，拟合得到精密的"指令-转速-推力"曲线。                                                                                                                                 |
|                                                                                                                                                                                                                                      |
| - 驱动逻辑优化：在嵌入式框架底层的电机类中内置非线性补偿算法或查找表，确保高层下发的推力指令能产生物理意义上的一致输出，从底层提升运动控制的确定性。                                                                                 |
